üìã Quick Start Guide (for Beginners)

üöÄ 2-Minute Setup

Follow these 2 simple steps to add curtain effect to your website:

  • Step 1: Copy-paste the code (curtain works automatically)
  • Step 2: Customize colors, text, speed (optional)

1. Download Required Files

<!-- Include CSS --> <link rel="stylesheet" href="path/to/curtain-effect.css"> <!-- Include Font Awesome for icons --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css"> <!-- Include JavaScript --> <script src="path/to/curtain-effect.js"></script>

2. STEP 1: Add Basic Curtain Effect (Copy-Paste Ready)

<!-- Add this to your HTML file BEFORE closing </body> tag --> <!-- Curtain Effect CSS --> <link rel="stylesheet" href="curtain-assets/css/curtain-effect.css"> <!-- Curtain Effect JavaScript --> <script src="curtain-assets/js/curtain-effect.js"></script> <!-- Basic Configuration (Required) --> <script> // Set curtain configuration window.curtainConfig = { title: 'Welcome to My Website', subtitle: 'Click to explore amazing content', buttonText: 'Enter Now' }; </script>
‚úÖ That's it! Your curtain effect is ready.
Just add the above code to your HTML file and the curtain will automatically appear when page loads.

3. STEP 2: Customize Your Curtain (Optional)

üìù Want to customize? Simply modify the window.curtainConfig object:
<!-- Replace the basic configuration with this customized version --> <script> window.curtainConfig = { // Basic Content title: 'IISF 2025', subtitle: 'India International Science Festival', buttonText: 'Enter Festival', // Theme Options: 'default', 'royal', 'elegant', 'gold', 'saffron' theme: 'saffron', // Animation Speed (1000 = 1 second, 5000 = 5 seconds) speed: 2000, // Effects sparkles: true, // Show falling sparkles sound: true, // Play clapping sound // Auto Opening autoOpen: false, // Set true to auto-open autoOpenDelay: 3000, // Wait 3 seconds before auto-open // Custom Images (Optional) leftCurtainImage: 'path/to/left-curtain.jpg', rightCurtainImage: 'path/to/right-curtain.jpg', // Callback Functions onComplete: function() { console.log('Welcome to the website!'); } }; </script>

4. Alternative Method: HTML Data Attributes

üí° Advanced users can also use HTML attributes instead of window.curtainConfig:
<!-- Add this div anywhere in your HTML body --> <div data-curtain-auto="true" data-curtain-title="My Website" data-curtain-subtitle="Welcome!" data-curtain-button="Enter" data-curtain-theme="saffron" data-curtain-speed="2000" data-curtain-sparkles="true" data-curtain-sound="false"> </div>

5. Available Options & What They Do

title: 'Welcome' - Main heading text displayed on curtains
subtitle: 'Click to Enter' - Secondary descriptive text
buttonText: 'Open Curtains' - Text shown on the action button
theme: 'default' | 'royal' | 'elegant' | 'gold' | 'saffron' - Visual style
autoOpen: false - Set true to auto-open without user interaction
autoOpenDelay: 5000 - Delay in milliseconds before auto-opening
speed: 2000 - Animation duration in milliseconds (1000-10000)
sparkles: true - Enable floating particle effects
sound: false - Play clapping sound during opening
leftCurtainImage: '' - Path to custom left curtain image
rightCurtainImage: '' - Path to custom right curtain image
onOpen: null - Callback function when curtains start opening
onComplete: null - Callback function when animation completes

6. Callback Functions

CurtainEffect.init({ onOpen: function() { console.log('Curtains are opening!'); // Your code here }, onComplete: function() { console.log('Animation completed!'); // Your code here } });

8. CSS Structure & Customization

/* Main curtain container - Full viewport coverage */ .curtain-container { position: fixed; /* Covers entire screen */ z-index: 10000; /* Above all content */ width: 100vw; height: 100vh; } /* Individual curtain panels - Split screen effect */ .curtain-panel { width: 50vw; /* Each panel is half viewport width */ transition: transform 2s; /* Smooth opening animation */ background: linear-gradient(...); /* Theme-based colors */ } /* Content box - Centered overlay with title/button */ .curtain-content-box { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); /* Perfect centering */ z-index: 10002; /* Above curtain panels */ } /* Theme classes - Override default colors */ .curtain-theme-saffron .curtain-panel { background: linear-gradient(45deg, #FF8C00, #FF7F00, #FF6347, #FFA500, #FFD700, #E68900); }

9. Manual Control & Advanced Usage

// Open curtains programmatically CurtainEffect.forceOpen(); // Change content dynamically CurtainEffect.setTitle('New Title'); CurtainEffect.setSubtitle('New Subtitle'); CurtainEffect.setButtonText('New Button Text'); CurtainEffect.setTheme('saffron'); // Change theme CurtainEffect.setCurtainImages('left.jpg', 'right.jpg'); // Check status if (CurtainEffect.isOpened()) { console.log('Curtains are open'); } // Clean up and remove CurtainEffect.destroy();
